# calculator_lmu

Calculateur de Carburant pour Le Mans Ultimate

## Objectif
- Pouvoir calculer la consomation de carburant ou d'énergie en plein milieu d'une endurance.
- Outils pour ingénieur ou pilote ne pilotant pas.
- Code en Java.

## Note d'Observation

Je test mon programme parfois en situation réelle durant de vrai endurance que je fais puis je compare le résultat avec mes calculs et le déroulé de la course. Les résultats de mes observations sont contenu dans le fichier ```Observation``` et sont rédigée en Markdown.
